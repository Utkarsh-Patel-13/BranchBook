# Root Project

The root `package.json` and config files govern the entire monorepo.

## Workspaces

- `apps/*` — Applications (web, server, native, docs)
- `packages/*` — Shared packages

## Tooling

- **Turbo** — Task orchestration
- **Ultracite** — Linting & formatting (Biome)
- **Husky** — Git hooks
