# @branchbook/db

Prisma client and schema. PostgreSQL via Prisma.

## Modules

- [Schema](/docs/packages/db/schema) — Workspace, Node, Note, Message models

## Commands

- `db:push` — Push schema to DB
- `db:migrate` — Run migrations
- `db:studio` — Prisma Studio
- `db:start` — Docker PostgreSQL
