# Getting Started

## Prerequisites

- [Bun](https://bun.sh) 1.3+
- Node.js 20+

## Development

```bash
# Install dependencies
bun install

# Run development servers
bun run dev

# Documentation site
bun run dev:docs
```

## Project Commands

From the root:

- `bun run dev` — Start all apps
- `bun run build` — Build all packages and apps
- `bun run dev:docs` — Start documentation site
- `bun run check` — Lint and format (Ultracite)
