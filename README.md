# Tech Stack

## Frontend

### Web
- React with TanStack Router
- TailwindCSS + ShadCN UI
- React Flow
- Vercel AI SDK
- Vercel AI Elements UI library (chat components)
- Lexical (rich text editor)
- Zustand (state management)
- React Query (server state)
- Framer Motion (animations)

### Mobile/Tablets
- React Native + Expo
- React Native equivalents of web libraries

### Shared
- tRPC (type-safe API)
- Zod (schema validation)
- date-fns (date handling)

## Backend
